# node-testing-demo
Repo of sample code for tutorial on node testing

```bash
# setup
npm install

# build
npm run build

# test commands
npm run test-js
npm run test-built
npm run test-es6
npm run test-xunit-report

npm run debug-test-js
npm run debug-test-built
npm run debug-test-es6
```

See "scripts" in `package.json` for available demo commands.
